# 如何创建 Hytale 账户

> 原文链接：[https://support.hytale.com/hc/en-us/articles/44134044725787-How-to-Create-a-Hytale-Account](https://support.hytale.com/hc/en-us/articles/44134044725787-How-to-Create-a-Hytale-Account)<br />
> 上次更新时间：2026/01/07 18:52

要游玩 Hytale，你需要在我们的网站上创建一个账户并注册你的用户名。这还将允许你管理个人资料、设置，并使用我们的商店。

## 在 Hytale 注册

请先访问我们的网站：[https://hytale.com/](https://hytale.com/)。

![](https://support.hytale.com/hc/article_attachments/44212556616603)

你需要点击 “Register”（注册）按钮，或直接点击此链接：[https://accounts.hytale.com/registration](https://accounts.hytale.com/registration)。

你可以使用应用进行注册，例如你的 Google 账户，或使用电子邮箱地址进行注册。

![](https://support.hytale.com/hc/article_attachments/44300101097243)

请确保在注册时可以访问该电子邮箱地址，并选择一个安全的密码。要了解更多关于账户安全的信息，请访问我们的文章：[《如何保障你的 Hytale 账户安全》](/player-support/accounts/how-to-keep-your-hytale-account-secure.md)。

![](https://support.hytale.com/hc/article_attachments/44300101098523)

完成以上步骤后，请输入你的年龄以继续注册流程。请注意，你必须年满 13 岁才能注册。

![](https://support.hytale.com/hc/article_attachments/44300101099803)

接下来，请完成 CAPTCHA 验证。

![](https://support.hytale.com/hc/article_attachments/44300089773979)

随后，你需要阅读并接受我们的 [隐私政策](https://hytale.com/privacy) 和 [服务条款](https://hytale.com/terms-of-service) 才能继续。

![](https://support.hytale.com/hc/article_attachments/44300089776795)

你需要通过点击电子邮件中提供的链接来验证你的账户。请注意，接收邮件可能需要几分钟时间。如果你尚未收到，请检查你的垃圾邮件文件夹。如有需要，你可以点击 “Resend email”（重发邮件）以获取一封新的邮件。

![](https://support.hytale.com/hc/article_attachments/44300089778203)

收到邮件后，请点击 “Confirm account"（确认账户）。

![](https://support.hytale.com/hc/article_attachments/44300089780507)

随后，你将被重定向至包含你的账户详情的页面。

![](https://support.hytale.com/hc/article_attachments/44300089781531)

如果你在创建过程中遇到任何问题，或有与账户相关的疑问，欢迎随时联系我们的支持团队。

## 如何使用并领取你预留的用户名

部分玩家通过我们的新闻通讯获得了一个预留令牌。要领取预留的用户名，你需要先通过创建一个新的个人资料来完成购买流程。完成后，你就可以选择你的预留用户名。

你可以使用预留的用户名，或选择一个不同的用户名。你也可以调整大小写。准备就绪后，点击“Apply reserved username”。

:::info[注意：]

未领取的预留用户名将在 2026 年 2 月初释放。

:::

![](https://support.hytale.com/hc/article_attachments/45164901276699)

如果你在此方面需要进一步的帮助，欢迎随时联系我们的支持服务。
