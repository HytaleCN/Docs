# 如何举报 Discord 规则违规者

> 原文链接：[https://support.hytale.com/hc/en-us/articles/43948452847387-How-to-Report-Discord-Rule-Breakers](https://support.hytale.com/hc/en-us/articles/43948452847387-How-to-Report-Discord-Rule-Breakers)<br />
> 上次更新时间：2026/01/06 05:23

Hytale 支持团队无法处理举报。目前，我们的版主团队仅能处理发生在我们官方平台——Hytale Discord 服务器——上的相关事件举报。

如果问题发生在 Discord 服务器之外，请按照事件发生的平台规则和举报流程，直接在对应平台进行举报。

## Hytale Discord

**`/report` 命令**

在我们的 Discord 服务器中进行举报时，请务必根据你的具体情况，使用最合适的举报方式。

![](https://support.hytale.com/hc/article_attachments/43948452840475)

随后，你需要在用户类别中填写用户名以选择对应的用户。你可以通过点击用户框来完成此操作。

接着，你需要根据该玩家的行为选择相应的违规选项。只需点击原因字段，并从以下选项中选择即可：

![](https://support.hytale.com/hc/article_attachments/43948452842395)

为了让社区能够更好地理解，各个举报原因的含义如下：

- Spam and Advertising（垃圾信息与广告）：用户发送重复消息、表情或反应，或在 Discord 中宣传第三方网站、服务器、Discord 名称/标签，或社交媒体。
- Inappropriate Chat Behavior（不当聊天行为）：用户在 Hytale Discord 频道中发送不当或具有冒犯性的内容。
- Inappropriate Profile Content（不当个人资料内容）：用户使用了不当或具有冒犯性的 Discord 标签、状态、个人简介、横幅、名称或头像。
- Player Harassment（玩家骚扰）：用户试图进行人肉搜索、DDOS 骚扰或敲诈勒索他人，分享个人信息，或发送恶意链接。
- Underage（未成年）：用户在 Hytale Discord 频道中明确表示其年龄未满 13 岁。
