## 截图软件

**Windows 与 Mac 内置截图说明**

Windows

- 按下 “PrintScreen” 键可截取整个屏幕。
- 按下 “Windows 键 + Shift + S” 可截取选定区域。
- 在需要的位置从剪贴板粘贴截图。

Mac

- 按下 “Shift + Command + 3” 可截取整个屏幕。
- 按下 “Shift + Command + 4” 可截取选定区域。
- 截图默认会保存到桌面。

**Hytale 内置截图工具**

- 在 Hytale 中按下 F12 即可进行截图。
- 将截图上传至图像分享网站。

**推荐的第三方截图软件**

Hytale 团队建议使用以下第三方程序进行截图和上传：

- [ShareX](https://getsharex.com/)
- [Lightshot](https://app.prntscr.com/)
- [Gyazo](https://gyazo.com/)
- [Imgur](https://imgur.com/upload)
- [Medal.tv](https://medal.tv/)

## 录制软件

**Open Broadcaster Software (OBS)**

- 一款免费的开源软件，用于视频录制和直播。
- 提供丰富的自定义选项，并兼容多个平台。
- [下载 OBS](https://obsproject.com/)

**Bandicam**

- 一款轻量级且易于使用的录制软件，可捕捉高质量的视频和截图。
- 提供多种功能，例如实时绘制和定时录制。
- [下载 Bandicam](https://www.bandicam.com/)

**Nvidia ShadowPlay**

- 适用于 Nvidia 显卡用户，能够在几乎不影响性能的情况下提供高质量录制。
- [了解更多关于 ShadowPlay](https://www.nvidia.com/en-us/geforce/geforce-experience/shadowplay)

**AMD ReLive**

- 适用于较新 AMD 显卡用户的替代方案，与 Nvidia ShadowPlay 功能相当。
- [了解更多关于 ReLive](https://www.amd.com/en/technologies/software-gaming-media)

## 视频上传位置

我们可以审核来自以下网站的视频上传内容：

- [YouTube](https://youtube.com/)
- [Vimeo](https://vimeo.com/)
- [Twitch](https://twitch.tv/)
- [Kick](https://kick.com/)
- [Reddit](https://reddit.com/)

如果视频不是公开可见的，请确保将其设置为 “unlisted”（不公开）。如果设置为 “private”（私密），我们的团队将无法查看。
