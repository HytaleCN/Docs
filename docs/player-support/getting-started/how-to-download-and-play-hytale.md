## 硬件要求

Hytale 目前支持 Windows、Mac 和 Linux。更多详情请参阅我们的 **[Hytale 硬件要求](https://hytale.com/news/2025/12/hytale-hardware-requirements)** 页面。

## 在哪里下载 Hytale

你可以通过我们的网站下载 Hytale：[https://accounts.hytale.com/download](https://accounts.hytale.com/download)。该页面会自动检测你当前使用的操作系统，但你也可以在页面底部选择其他系统。

出于安全原因，请始终通过 [Hytale 官方网站](https://hytale.com/) 下载 Hytale 启动器。有关账户安全的更多信息，请参阅我们的[**《如何保障你的 Hytale 账户安全》**](/player-support/accounts/how-to-keep-your-hytale-account-secure.md)文章。

![](https://support.hytale.com/hc/article_attachments/45306018632219)

## 如何安装 Hytale 启动器

### Windows

下载 Hytale 启动器后，双击以启动安装程序。点击“Next”，选择你的安装文件夹，然后选择你的游戏文件夹并点击“Install”。

安装完成后，点击“Finish”，Hytale 启动器将自动打开。

![](https://support.hytale.com/hc/article_attachments/45306136669595)

### macOS

下载适用于 macOS 的 Hytale 启动器后，点击该文件以打开安装对话框，然后将其拖入 Applications 文件夹。

如果你需要在 macOS 上安装应用的帮助，请查阅 Apple 关于 [在 Mac 上从互联网或光盘安装和卸载应用](https://support.apple.com/en-gb/guide/mac-help/mh35835/mac) 的指南。

![](https://support.hytale.com/hc/article_attachments/45306124468123)

### Linux

打开终端并运行以下命令：

`flatpak install hytale-launcher-latest.flatpak`

然后运行以下命令以启动启动器：

`flatpak run com.hypixel.HytaleLauncher`

你可能需要先安装 Flatpak。请查看你的 Linux 发行版文档以获取详细信息。

## 如何安装 Hytale

打开 Hytale 启动器并点击 Install 以下载并安装游戏。Hytale 将于 _2026 年 1 月 13 日_ 发布。

![](https://support.hytale.com/hc/article_attachments/45306420855963)

## 选择你的游戏配置文件

你可以在 Hytale 启动器右上角选择不同的游戏配置文件。

![](https://support.hytale.com/hc/article_attachments/45306124473115)

# 如何卸载 Hytale

根据你的操作系统，你可以选择分别或同时卸载 Hytale 与 Hytale 启动器。

## 卸载游戏

你可以直接通过 Hytale 启动器的设置面板卸载游戏——点击右上角的设置按钮并选择 Uninstall。

该面板中提供了许多选项，但要继续卸载，请再次选择 Uninstall 按钮。

![](https://support.hytale.com/hc/article_attachments/45306124471323)

你的游戏现已卸载。

## 卸载 Hytale 启动器

### Windows

在 Windows 上卸载启动器时，点击开始按钮，搜索“Hytale”，右键点击应用并选择 Uninstall。按照卸载向导操作，以选择是否保留你的游戏文件和数据。默认情况下，游戏和启动器都会被移除。

如需了解更多有关在 Windows 上移除应用的帮助，请参阅 Microsoft 关于 **[在 Windows 中卸载或移除应用和程序](https://support.microsoft.com/en-us/windows/uninstall-or-remove-apps-and-programs-in-windows-4b55f974-2cc6-2d2b-d092-5905080eaf98#id0ebd=windows_11)** 的指南。

![](https://support.hytale.com/hc/article_attachments/45306124472219)

### macOS

在 macOS 上卸载启动器时，前往 Applications 文件夹，然后右键点击 Hytale 启动器并选择 Move to Trash。如需了解更多有关在 macOS 上删除应用的帮助，请查阅 Apple 关于 **[在 Mac 上删除或卸载应用](https://support.apple.com/en-us/102610)** 的指南。

![](https://support.hytale.com/hc/article_attachments/45305997497755)

### Linux

打开终端并运行以下命令以卸载 Hytale：

`flatpak uninstall com.hypixel.HytaleLauncher`
