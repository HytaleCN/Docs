# 如何加入 Hytale 官方 Discord 服务器

:::tip[提示：]

这篇文章来自 Hytale 官方，我们仅对原文进行了翻译处理。

> 原文链接：[https://support.hytale.com/hc/en-us/articles/45314940049563-How-to-Join-the-Official-Hytale-Discord-Server](https://support.hytale.com/hc/en-us/articles/45314940049563-How-to-Join-the-Official-Hytale-Discord-Server)<br />
> 上次更新时间：2026/01/11 18:38

:::

**Hytale 官方 Discord 邀请链接为：[https://discord.gg/hytale](https://discord.gg/hytale)。**

Hytale Discord 是 Hytale 社区的核心枢纽，玩家可以在这里获取最新的 Hytale 新闻、分享想法，并展示自己正在制作的内容。

:::info[注意：]

Hytale 官方 Discord 服务器**绝不会**向你索要任何 Hytale 账户信息。请警惕主动联系你的私信，以及声称自己是 Hytale 工作人员的非官方服务器或用户，并考虑关闭来自陌生用户的私信功能。

:::

## 如何加入 Hytale 官方 Discord

要加入 Hytale 官方 Discord 服务器，请按照以下步骤操作：

1. **创建或登录 Discord 账户**

   如果你还没有 Discord 账户，可以前往 [https://discord.com/](https://discord.com/) 创建一个。

   ![](https://support.hytale.com/hc/article_attachments/45314923926939)

2. **使用官方邀请链接并接受邀请**

   点击上方列出的 Hytale 官方 Discord 邀请链接。这将会在浏览器或桌面应用中打开 Discord。当系统提示时，选择 Accept Invite 以加入服务器。

   ![](https://support.hytale.com/hc/article_attachments/45314940049051)

3. **完成服务器入门流程**

   加入 Discord 服务器后，你可能需要回答一些问题并查看服务器规则，才能访问所有频道。完成这些步骤后，你即可访问 Hytale 官方 Discord 服务器！

   ![](https://support.hytale.com/hc/article_attachments/45314940049435)
